更新手順

  1. この画面で歌枠追加、またはキー・ジャンル同期を実行する。
  2. 必要に応じてAPIステータスと確認項目を再チェックする。
  3. 「静的データ生成を開始」を押してGitHub Actionsを起動する。
  4. Actions完了後、Cloudflare Pagesのデプロイが終わると公開サイトへ反映されます。

このページをスマホ単独で使う場合は、Cloudflare Accessで保護し、D1とGitHub用のSecretをCloudflare Pagesに設定してください。
