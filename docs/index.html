Guide

使い方

曲を探す、歌枠をたどる、配信用の選曲を作るための歌唱データベースです。

1 データを見る

Mainを切り替えて、見たいデータだけに絞ります。

2 曲を探す

全曲リスト・ランキング・検索で曲名・アーティスト・ジャンルから探せます。

3 歌枠を見る

タイムラインから歌枠を選んで、その場で動画を再生できます。

データ範囲

  • ヘッダーの Main は、表示する歌唱データの範囲を切り替えます。
  • ヘッダー右の 公式アイコン（YouTube、X ロゴ）から公式チャンネルの情報や X へ移動できます。
  • リスナーモードは閲覧向け、配信者モードは選曲・セトリ作成向けです。

曲を探す

  • / または Ctrl+K でサイト内全検索を開き、曲・アーティスト・配信を横断検索できます。
  • 全曲リストではジャンル・タグ・並び替えで詳細な絞り込みができます。
  • 曲を押すと 歌唱回数・キー・初披露日・歌枠履歴 を確認できます。

動画ビューワー（歌枠・歌みた/オリ曲）

  • タイムラインや歌みた・オリ曲のサムネイルを押すと、その場で動画を再生できます。
  • プレイヤー下の操作バーで 前へ / 再生・一時停止 / 次へ / 連続再生 / リピート / 栞（プレイリスト保存） が使えます（歌枠・動画どちらも共通）。
  • Space で再生・一時停止、← → で 10秒の巻き戻し・早送りができます。
  • 歌枠ではセットリストの メモ で再生位置をタイムスタンプ記録、バッジをタップでその箇所へジャンプ。共有バッジ はコミュニティ審査済みのタイムスタンプで、「+ 提案」で投稿できます。
  • 共有 ボタンで、いま見ている位置付きのリンクをコピー・SNS共有できます。
  • 戻るとミニプレイヤーで再生を継続。ミニプレイヤーのタイトルをタップで元のビューワーへ戻れます。音楽プレイヤーで聴く で画面下の音楽バーへ移せます。

音楽プレイヤーバー

  • 画面下のバーで 前へ / 再生・一時停止 / 次へ / 連続再生 / 1曲リピート / 全体リピート / シャッフル を操作できます。
  • リピートの隣の キューアイコン で「次に流れる曲」の一覧を開き、曲をタップして移動できます。
  • サムネ右上のボタンや 動画で見る から、いつでも動画ビューワーへ切り替えられます。

プレイリスト（マイリスト）

  • マイリストを作成し、歌枠と動画（歌みた・オリ曲）を混在させて追加できます。再生すると順番にビューワーで連続再生されます。
  • タイムライン・歌枠一覧・動画ビューワーの 栞ボタン、歌みた・オリ曲タブの「☑ 選択」での まとめて追加 から保存できます。
  • 行は ドラッグで並び替え、リンクを共有 で他の人と共有、YouTubeで連続再生 で YouTube 上の連続再生も可能です。

ランキング・配信者モード

  • ランキングは今月・先月・直近7日・月別に切り替えられ、前月比も確認できます。
  • 配信者モードでは選曲候補をセトリに追加して並び替え・コピーができます。

キーボードショートカット

  • / Ctrl+K — サイト内全検索を開く
  • T — ダーク/ライトテーマ切替
  • ? — このヘルプを開く
  • Esc — 検索・モーダルを閉じる
  • Space — 動画ビューワーで 再生 / 一時停止
  • ← → — 動画ビューワーで 10秒 巻き戻し / 早送り

うまく表示されないとき

  • データ更新直後は数分待ってから再読み込みしてください。
  • 古い表示が残る場合はブラウザのハードリロードを試してください。
  • 曲名やキーの誤り・不具合はフッターの「お問い合わせ」から連絡できます。
まずは範囲とモードを選べます 閲覧はリスナー、選曲やセトリ作成は配信者モード。詳しい使い方はヘッダーの「?」から見られます。
